# Xangô E-Racing - Otimização

## Branches

### ```latest/car-optimization```

Essa branch contém o código _funcional_ mais atual para realizar a otimização dos parâmetros do carro. O conteúdo aqui já foi testado e está 100% funcional.

### ```dev/car-optimization```

Essa branch contém o código _em desenvolvimento_ para realizar a otimização dos parâmetros do carro. O código aqui está em processo de desenvolvimento, não foi completamente testado e portanto pode apresentar erros.

### ```latest/race-line-optimization```

Essa branch contém o código _funcional_ mais atual para realizar a otimização do traçado na pista. O conteúdo aqui já foi testado e está 100% funcional.

### ```dev/race-line-optimization```

Essa branch contém o código _em desenvolvimento_ para realizar a otimização do traçado na pista. O código aqui está em processo de desenvolvimento, não foi completamente testado e portanto pode apresentar erros.

## Setup

### Instalar dependências

- Se estivar usando venv, rode: ```pip install -r requirements.txt```
- Se não, rode: ```python3 pip install -r requirements.txt```

### Iniciar otimização

- Se estivar usando venv, rode: ```python start.py```
- Se não, rode: ```python3 start.py```
